Personal Portfolio Website

This is a personal portfolio website developed using React JS.  
The website showcases my personal information, skills, projects, certifications, and contact details.

 Live Website
https://portfolio-abs-ebon.vercel.app/

Project Description
The portfolio website is designed to present my academic background, technical skills, and projects in a professional manner.  
It uses modern UI design principles such as glassmorphism, smooth animations, and responsive layouts to ensure a good user experience across all devices.

Tech Stack Used
- React JS
- Vite
- Tailwind CSS (for layout & responsiveness)
- Custom CSS (for animations and glassmorphism)
- HTML5
- JavaScript

 GitHub Repository
 https://github.com/Abeer-m-s/Portfolio

Features
- Responsive design (mobile, tablet, desktop)
- Animated introduction section
- Glassmorphism UI components
- Projects timeline
- Skills and certifications display
- Contact information section
